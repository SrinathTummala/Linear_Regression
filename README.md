## Work Flow
* Data Loading and Understanding
* Data Visualization
* Data Preparation - Split into test,train and Rescale
* Data Modelling
* Residual Analysis, Checking Assumptions of Linear Regression
* Prediction and Evaluation on the Test Dataset
